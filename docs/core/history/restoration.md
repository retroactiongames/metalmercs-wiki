# The Restoration

The Restoration was the long process by which the Great Houses reestablished stable government, trade, law, and military protection across what is now called the Core.

## Overview

To most Core citizens, this era marks the rebirth of civilization. To critics, it marks the creation of a fortified remnant that mistook survival for recovery.
