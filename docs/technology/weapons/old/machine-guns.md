# Machine Guns

Machine guns are light ballistic weapons used for anti-infantry work, close defense, and suppression of unarmored targets.

## Battlefield Use

Against heavy mechs they are rarely decisive, but they remain useful against vehicles, infantry, drones, and exposed systems.
