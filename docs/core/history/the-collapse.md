# The Collapse

The Collapse was not a single event but a cascading failure of Imperial authority, infrastructure, communication, logistics, and trust.

## Overview

Worlds were cut off. Trade routes failed. Battlestars vanished. Technologies became impossible to maintain. In many places, civilization survived only locally.
